# Vo Duc Hieu — Portfolio

A Visual Studio Code themed developer portfolio built with **Next.js** and deployed on **Vercel**.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: CSS Modules
- **Language**: TypeScript
- **Deployment**: Vercel

## Getting Started

```bash
npm install

cp .env.example .env

npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Environment Variables

| Variable | Description |
|----------|-------------|
| `NEXT_PUBLIC_GITHUB_USERNAME` | GitHub username for the profile page |
| `GITHUB_API_KEY` | GitHub Personal Access Token (no scopes needed) |
| `DEV_TO_API_KEY` | DEV.to API key (optional, for articles) |
